CURSO DE CSS

Selectores

  • Universal: Se pone un asterisco * que selecciona TODOS los elementos de la página. Es por este selector que toda la página tiene fuente sans-serif.
  • De tipo: Selecciona todos los elementos de un mismo tipo (es decir, todos los h1, todos los p, todos los input, etc).
  • Clases: Selecciona todos los elementos de una misma clase.
  • ID: Selecciona el elemento con el ID especificado.
  • Atributo: En una determinada etiqueta, creas un atributo con el nombre que vos quieras y la podes seleccionar, como si fuera un ID.
  • Descendiente: Sirve para seleccionar un elemento dentro de otro (por ejemplo, podría seleccionar todos los b dentro de un parrafo p; también podría seleccionar todos los )
  • Pseudoclase: LO VEMOS MÁS ADELANTE, TOCÁ ACÁ PARA IR A ESA SECCION



Especificidad

Cuando hacemos una página, a veces se suelen mezclar los estilos que escribimos (por ejemplo, a un mismo parrafo le damos dos colores). Por eso es importante saber cuáles son los niveles de autoridad de cada selector, de manera que podamos elegirlos sabiamente.




Unidades de medida

Cuando, por ejemplo, queremos establecer el tamaño de letra de un título (font-size), tenemos dos opciones de unidades: las fijas (no dependen de nada, como los px, cm o mm) las relativas (dependen de cosas como el tamaño de la pantalla, el tamaño del contenedor, etc). Algunos ejemplos son los em (1em=16px) y los viewport (hay de dos tipos: viewport width o vw, y viewport height o vh)

El div azul ocupa la mitad de la pantalla porque tiene 50vw, y aunque se modifique el tamaño de la ventana, el div seguirá ocupando ese espacio.




Propiedades de texto

  • Font-size, tamaño de letra
  • Font-family, es la tipografía (osea, el tipo de letra)
  • Line-height, es el espacio que ocupa un determinado texto (NO es lo mismo que el font-size; el line-height incluye también el espacio vacío arriba y abajo de la letra). Por ejemplo, Este texto tiene line-height=1
    y este otro tiene line-height=4
  • Font-weight, es el grosor de la letra
  • Text-align, con el valor center hace que el texto se coloque en el medio de su contenedor
  • Font-variant, no lo explicó pero lo guardo
  • Font-style, no lo explicó pero lo guardo
  • Text-decoration, no lo explicó pero lo guardo
  • List-style, no lo explicó pero lo guardo



Propiedades de cajas

  • Background-color, cambia el color de fondo
  • Padding, es la distancia entre un contenido (texto, imagen, título, etc) y las paredes de su contenedor. Se puede establecer el padding de cada borde (top, right, bottom, left) uno por uno (padding-top, padding-right, padding-bottom, padding-left) o en una sola línea, pudiendo escribir 2, 3 o 4 valores:
    • padding: top right bottom left ;
    • padding: top ejeX bottom ;
    • padding: ejeY ejeX ;
  • Margin, es la distancia entre dos cajas. Para establecerlo, funciona igual que el padding
  • Border, sirve para caracterizar el borde de una caja. Se usan los siguientes parámetros:
    • border: tamaño tipo color ;
    • border: 10px solid blue ;
  • Border radius, establece cuántos píxeles vamos a eliminar para que un cuadrado tenga bordes redondos
